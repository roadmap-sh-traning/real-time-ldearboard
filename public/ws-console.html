Game ← server: player.joined, player.left, score.updated (includes newScore), error  |  Leaderboard ← server: snapshot (entries[]: rank, playerId, name, score)  |  Game → server: join, score, penalty-kick, leave  |  Penalty ← server: penalty-kick.result
